CodeSystem: Cochrane PICO Ontology

Official URL: http://data.cochrane.org/ontologies/pico/ Version: 4.0.1
Draft as of 2025-03-04 Responsible: Cochrane Computable Name: CochranePICOOntology

The PICO ontology describes questions in evidence based healthcare with the intention of supporting the publishing of evidence as linked data.

This Code system is referenced in the content logical definition of the following value sets:

  • CochranePICO
  • CochranePICOIntervention
  • CochranePICOOutcome
  • CochranePICOPopulation
