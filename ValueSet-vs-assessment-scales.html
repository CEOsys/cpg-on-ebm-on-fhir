Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include codes from http://snomed.info/sct where concept is-a 273249006 (Assessment scales)
  • Include codes from http://snomed.info/sct/731000124108 where concept is-a 273249006
  • Include these codes as defined in http://loinc.org
    CodeDisplay
    72514-3Pain severity - 0-10 verbal numeric rating [Score] - Reported
    54627-5Signs and symptoms of delirium (from CAM)
  • Include codes from http://loinc.org where PROPERTY = Score
