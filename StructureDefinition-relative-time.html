Extension: RelativeTime

Official URL: https://www.netzwerk-universitaetsmedizin.de/fhir/cpg-on-ebm-on-fhir/StructureDefinition/relative-time Version:
Draft as of 2025-03-04 Computable Name: RelativeTime

Timing in which the characteristic is determined.

Context of Use

This extension may be used on the following element(s):

  • Element ID EvidenceVariable.characteristic
  • Element ID PlanDefinition.action.timing[x]
  • Element ID ActivityDefinition.timing[x]
  • Element ID Timing

Guidance

This extension allows the use of the FHIR R6 RelativeTime Datatype in the context of the CPG-on-EBM-on-FHIR project.

The extension can be used in the following places:

  • RecommendationEligibilityCriteria.characteristic
  • RecommendationAction
  • RecommendationPlan.action

Usage info

Usage:

  • Use this Extension: Eligibility Criteria, Recommendation Action and Recommendation Plan
  • Examples for this Extension: RecommendationPlanNestedCombination
